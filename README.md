# Alzheimer's Disease Detection Backend (PHP)

A pure PHP backend API for a healthcare application focused on early detection and management of Alzheimer’s disease using artificial intelligence.

The system provides AI-based diagnosis with four severity levels and creates a supportive environment by connecting patients with their family members and caregivers. It also includes memory training features through reminders and family events.

---

## Overview

This backend acts as the core of a cognitive healthcare platform that combines medical diagnosis with daily life support.  
It helps families and caregivers stay connected with patients while assisting in memory training and patient monitoring.

---

## Key Features

- AI-based Alzheimer’s disease detection with four severity levels
- Secure patient profile management
- Linking patients with family members and caregivers
- Adding and managing family members
- Memory training through reminders and daily notifications
- Family events and occasions to stimulate memory recall
- RESTful API with JSON responses
- Designed for mobile healthcare applications

----

## AI Diagnosis

The system processes medical data such as brain imaging and clinical inputs.  
The AI model classifies Alzheimer’s disease into four stages, providing:
- Diagnosis level
- Confidence score

Results are returned in a structured JSON format for easy integration with client applications.

---

## Memory Training & Family Support

The platform supports cognitive training by:
- Sending reminders for daily tasks
- Adding family events and special occasions
- Encouraging patient interaction with familiar people and memories

Family members can monitor progress and support the patient through the application.
